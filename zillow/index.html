Brand Voice & Strategic Narrative

Zillow's voice in agent marketing is outcome-focused and practical: agents want to make money and save time. Post-NAR, the winning agent positioning connects products directly to agent economics—how Showcase sells listings faster and for more, how Premier Agent acquires quality leads, how Follow Up Boss and ShowingTime+ close deals faster. For the PMM function, everything organizes around a single thesis:

Central Narrative
The Agent Economics Reset

The NAR settlement fundamentally changed how agents value partnerships. Agents now need to justify every dollar spent. The platforms that connect lead generation, listing marketing, CRM, and transaction tools into a unified system—and prove ROI across the entire agent lifecycle—will own the agent market. Zillow's integrated portfolio makes this possible.

Connect to Agent Economics
Shift from features to business outcomes
Every message maps to how agents make and save money: "Showcase sells faster and for more," "Premier Agent qualified leads," "Follow Up Boss closes deals 30% faster." Proof > promises.
Speak Agent Language
Shift from PMM terminology to agent vocabulary
Agents think in leads, closings, time-to-close, and buyer/seller ratios. Not ARR, net expansion, or market share. Use their language to build trust.
Emphasize the Unified System
Elevate from point products to platform
Agents switching between Zillow for leads, Listing Mule for marketing, BoomTown for CRM, and Zipline for transactions waste hours. One platform + Zillow data = competitive advantage.
